# Claudtero

AI-Chatbot for exploring PDFs in Zotero.

## Requirements

- Python 3.9 or higher
- Node.js
- Zotero 7

## Installation

1. Install Python dependencies:
```bash
npm run install-py-deps
```

2. Start the application:
```bash
npm start
```

This will start both the Docling service and the Zotero plugin development server.

## Usage

Open a PDF in Zotero and use the Claudtero chat in the sidebar to ask questions about your document.
